Help for Students

Welcome to this exercizing platform, it is designed just for you!

General principle of the platform

To be able to practice, you must first register for a session (attention, each session has a start date and an end date, outside which exercises are not available).

Each session offers an ordered list of exercises. The first exercise is available as soon as the session opens. Other exercises may also be available straight away or only a few days later (in the case of a timed) session or when you reach a certain score to the previous exercise.

There are skills associated with each exercise. These are not yet visible put will be in some time (missing features - help wanted!).

Solving an exercise

All exercices work in the same way:

  1. they display a statement containing instructions and an archive file of .tar.gz type to download;
  2. You download the archive and puts it on a Unix system [1] ;
  3. you extract the content of the archive on this system and work with it, guided by the questions in the statement;
  4. you in turn create an archive containing the result of your work;
  5. you put this archive on the exercise page, which analyzes it and tells you what's correct or not in your answers. You also get a score (overall success percentage) for this attempt.

A Hints section is usually present below the questions, in case you're having a hard time solving the exercise. But try first to not look at it: you will progress more sustainably. Finding a solution is a very important part of the learning process, don't spoil yourself ;-)

You can try again solving an exercise as many times as you want (without having to leave the exercise page), as long as the exercise is available.

The system records your score for each attempt. If the session you are participating in is organized by a teacher, do not hesitate to ask if he/she only accounts your last attempt, or your best attempt ;-)

Remarks

[1] Where to solve an exercise?

  • on your machine from a terminal: if you have a Linux, FreeBSD or MacOS operating system then you have a Unix-like system! If you have Windows, it offers you to install Unix (Ubuntu) directly as a subsystem inside Windows.
  • on a website from your browser: several sites provide you with free access to Unix (real or emulated) systems that allows files to be uploaded and downloaded (e.g., repl.it, CoCalc, ...). This is a little less comfortable but it allows you to work directly from your internet browser.
